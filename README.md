# Docker Demo Project

This is a simple Docker demo project that shows how to run an application using Docker containers.

## 📌 Project Description
This project demonstrates the basic usage of Docker including building images and running containers. It is useful for beginners who want to understand how Docker works.

## 🛠 Technologies Used
- Docker
- Git
- GitHub

## 📂 Project Structure
docker-demo/
│
├── Dockerfile
├── docker-compose.yml
├── README.md
└── application files

## 🚀 How to Run the Project

### Step 1: Clone the Repository
git clone https://github.com/your-username/docker-demo.git

### Step 2: Go to Project Folder
cd docker-demo

### Step 3: Build Docker Image
docker build -t docker-demo .

### Step 4: Run Docker Container
docker run -p 3000:3000 docker-demo

## 📖 Learning Purpose
This project is created for learning Docker basics and understanding containerization.
